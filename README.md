# AlgoritmoDeOrdenacao

Projeto desenvolvido em C# para demostrar os principais algorítimos de ordenação.

Algoritimos abordados:

- Insert Sort
- Selection Sort
- Bubble Sort
- Combo Sort
- Merge Sort
- Quick Sort
- Heap Sort
- Shell Sort
- Radix Sort
- Gnome Sort
- Counting Sort
- Bucket Sort
- Cocktail
